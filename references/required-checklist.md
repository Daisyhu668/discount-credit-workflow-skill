# 必填清单（贴现授信）

## 1. 业务前端必填

- 企业名称
- 统一社会信用代码
- 注册时间（YYYY-MM-DD）
- 注册地址
- 法定代表人
- 行业类型
- 申请日期（YYYY-MM-DD）
- 上一年度营业收入（万元）
- 净资产（万元）

## 2. 建议联网检索补齐（如客户未提供）

- 统一社会信用代码
- 注册时间
- 注册地址
- 法定代表人
- 行业类型

检索后写入 `--enrich-json`，再进入正式生成。

## 3. 自动计算字段（脚本生成）

- 申请日期+1年
- 上一年度营业收入*80%
- 净资产*12
- 授信额度（取两者孰低）

## 4. 生成前确认规则

- 缺失任一“业务前端必填”字段时，默认停止正式生成。
- 仅在人工明确同意时，允许 `--allow-missing` 强制继续。
- 生成前必须输出并确认 `01_字段确认单.md`。
