# 与代理协作的标准提词

## 固定约束

- 只允许改动 `discount-credit-workflow/` 目录
- 禁止改动原业务目录（例如 `线上贴/`）
- 输出命名必须遵循 `企业名称-文件名`

## 推荐提词模板

```text
请只修改 discount-credit-workflow 目录。
目标：完善贴现授信一键流程（确认单、授信材料、两封邮件）。
要求：
1) 先按企业名称做联网检索并补齐必填字段；
2) 先返回字段确认单，再生成材料；
3) 生成后发送两封邮件；
4) 全部输出文件名按“企业名称-文件名”。
最后给我：改动文件清单、执行命令、验证结果。
```

## 交付验收

- `run_discount_pipeline.py` 可以一次跑通
- 缺失必填字段时会阻断
- 至少输出两份授信报告
- 客户经理与客户两封邮件草稿存在
- 需要时可通过 SMTP 实际发送两封邮件
